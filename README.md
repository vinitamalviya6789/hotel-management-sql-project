# Hotel Management SQL Project

## Project Objective
This project analyzes hotel booking data using SQL to answer
business-related questions.

## Files Included
- hotel_schema.sql : database tables structure
- hotel_data.sql   : sample hotel data
- hotel_queries.sql: analysis queries

## SQL Concepts Used
- JOIN
- GROUP BY
- HAVING
- Subqueries
- Window Functions

## Key Analysis
- Total bookings
- Revenue analysis
- Repeat customers
- Room-wise performance

## Tools Used
- MySQL
- GitHub
